# Python for Dummies 

A collection of Python problems and solutions gathered from various books, solved for practice and learning.

## Books

This repository draws problems from:

- **[Impractical Python Problems](https://nostarch.com/impracticalpythonprojects)** — Lee Vaughan
- **[Automate the Boring Stuff with Python](https://automatetheboringstuff.com/)** — Al Sweigart
- **Python Puzzles** — Matthew Whiteside

##  References

- Random Problems (various online sources)
- Including Ai generated problems (Claude Ai and Chatgpt)

## About

This repo is a practice ground for working through exercises pulled from different Python books, rather than a single structured course. Expect a mix of topics and difficulty levels as new problems are added.
